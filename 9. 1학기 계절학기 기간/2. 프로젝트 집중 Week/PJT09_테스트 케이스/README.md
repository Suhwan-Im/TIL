# SW테스트 케이스 만들기
> :bulb: 1학기 관통 프로젝트에 대해 테스트 케이스를 도출하고 이를 수행해 보자

## 일일 과제 수행 순서

1. 관통 프로젝트에 대해 테스트할 기능에 대한 화면정의서를 준비하고 없다면 화면을 캡쳐하여 준비한다.
2. 각 기능에 대해 (또는 화면에 대해) 제어 흐름도를 작성한다. 
3. 제어 흐름도의 if 분기를 중심으로 결정테이블을 작성하여 논리 케이스를 만든다
4. 최종 케이스를 완성한다.
